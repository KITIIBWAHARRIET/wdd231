Small-Scale Poultry Farming Resource Hub

KARRY POULTRY FARM

Small-Scale Poultry Farming Resource Hub

This name was selected because it clearly represents a centralized online resource designed to help small-scale poultry farmers and beginners access reliable farming information, tools, and educational resources.

Site Purpose

The purpose of this website is to provide structured educational content for small-scale poultry farmers. The site will include information about poultry breeds, feeding programs, housing systems, disease prevention, and farm management techniques.

Interactive features will include dynamically loaded poultry breed data using JSON, modal dialogs displaying detailed information, filtering tools, saved user preferences using localStorage, and a contact or inquiry form.

Scenarios

  • Which poultry breed is best for egg production on a small farm?
  • What feeding program should I use for healthy poultry growth?
  • How can I prevent common poultry diseases?

Color Scheme

  • Dark Green (#2E7D32) – Used for headings, navigation, and accents.
  • Light Cream (#FFF8E1) – Used as the main background color.
  • Brown Accent (#6D4C41) – Used for borders and highlights.

Typography

  • Poppins – Used for headings.
  • Open Sans – Used for body text.
